# Econ Portfolio

This repository contains applied economics and data analysis projects completed during my undergraduate studies in Economics.

The projects focus on empirical analysis, econometrics, and time series forecasting, with an emphasis on using real-world data and clearly communicating results.

## Projects

### Research Seminar 
Independent empirical research project using Canadian provincial panel data.
- Fixed effects regression analysis
- Robustness checks and sensitivity analysis
- Data sourced from Statistics Canada
- Tools: Stata, Excel

### Forecasting & Time Series Analysis
Applied forecasting projects completed as part of an economics forecasting course.
- AR, MA, ARMA, and ARIMA models
- Model evaluation using RMSE and MAE
- Forecast interpretation and reporting
- Tools: R, Excel

## Tools & Skills
- Stata (panel data, regression analysis)
- R (time series forecasting)
- Excel (data cleaning, analysis, reporting)
- Data visualization and interpretation

---

This portfolio is intended to demonstrate applied analytical skills and independent problem-solving relevant to entry-level analyst, economics, and data-focused roles.
