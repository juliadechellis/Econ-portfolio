# Research Seminar: Union Density and Income Inequality

## Overview
This project examines the relationship between union density and after-tax income inequality across Canadian provinces from 2000–2023.

The analysis uses panel data methods to account for unobserved provincial and time-specific factors.

## Data
- Statistics Canada provincial-level data
- After-tax income inequality (Gini coefficient)
- Union density
- Macroeconomic control variables

## Methodology
- Fixed effects regression models
- Robustness and sensitivity checks
- Sub-sample analysis

## Tools
- Stata
- Excel

## Repository Structure
- `code/` – Stata do-files used for analysis
- `data/` – Cleaned datasets or data documentation
- `output/` – Regression tables and figures
- `paper/` – Final research paper (PDF)
